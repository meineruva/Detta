# Detta Shared Types

This package contains shared TypeScript interfaces and utilities for the Detta Attendance System.

## Usage

Import types from `@detta/shared`:

```typescript
import { AttendanceRecord, User } from '@detta/shared';
```

## Contribution

- Add new types to `src/types.ts`.
- Ensure all types are exported from `src/index.ts`.
- Run `tsc` to verify compilation before committing.
